### Type of Data
integar 정수 - numbers   
float - 소수점   
string - 텍스트 / “” 안에 꼭 써야됨 그래야만 텍스트로 인식함

const 절대 안변하는 변수   
let 나중에 변할 수 있는 변수   
Always const sometimes let never var

```
const a = 5;
const b = 2;
let myName = "claire";

console.log (a + b);
console.log (a * b);
console.log ("hello" + myName)

myName = "claire Kim";
console.log ("hello" + myName)
```
